Instalar Zenit en Windows

5 pasos · menos de 2 minutos · Windows 10 y 11

Descargar instalador para Windows

Antes de empezar: la alerta azul es normal

Windows mostrará una pantalla azul que dice “Windows protegió tu PC”. Aparece con todo programa nuevo de un desarrollador independiente que aún no paga una firma digital (un certificado que cuesta cientos de dólares al año). No significa que Zenit sea un virus. En el Paso 3 te mostramos exactamente qué botón tocar para continuar.

1

Descarga el instalador

Toca el botón de descarga. Se guardará el instalador de Zenit en tu carpeta de Descargas (un archivo .exe que empieza con Zenit-POS-Setup).

📸 Captura opcional: archivo .exe en la carpeta Descargas
(assets/img/win-1-descarga.png)
2

Abre el archivo descargado

Haz doble clic en el instalador que descargaste. Windows tardará un par de segundos en revisarlo.

📸 Captura opcional: doble clic en el instalador
(assets/img/win-2-abrir.png)
3

En la alerta azul, toca “Más información”

Aparecerá esta pantalla. No toques “No ejecutar”. Haz clic en el enlace “Más información”:

🛡️ Windows protegió tu PC
Windows protegió tu PC

Microsoft Defender SmartScreen impidió el inicio de una aplicación no reconocida. Ejecutar esta aplicación puede poner en riesgo tu PC.

Más información
No ejecutar

💡 Al inicio solo aparece el botón “No ejecutar”. El botón para continuar se muestra después de tocar “Más información”.

4

Toca “Ejecutar de todas formas”

Al tocar “Más información” aparece un nuevo botón abajo. Haz clic en “Ejecutar de todas formas”:

🛡️ Windows protegió tu PC
Windows protegió tu PC

Microsoft Defender SmartScreen impidió el inicio de una aplicación no reconocida. Ejecutar esta aplicación puede poner en riesgo tu PC.

Aplicación: Zenit-POS-Setup-1.5.0.exe
Editor: Editor desconocido
Ejecutar de todas formas No ejecutar
5

Sigue el asistente y ¡listo!

Se abrirá el instalador de Zenit. Acepta y espera unos segundos. Al terminar, Zenit se abrirá solo y podrás crear tu cuenta o entrar en modo local. 🎉

📸 Captura opcional: instalador de Zenit finalizado
(assets/img/win-5-listo.png)

A partir de ahora Zenit se actualiza solo: no tendrás que repetir este proceso.
